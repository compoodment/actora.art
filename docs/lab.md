# Lab

Short descriptions of the experiments and unfinished things at [actora.art/lab](https://actora.art/lab/).

- [Space](https://actora.art/lab/space/) — a 3D Newtonian orbital sandbox with selectable Solar System and Alpha Centauri simulations.
- [Liminal](https://actora.art/lab/liminal/) — an explore-only first-person walker in a concrete test chamber.
- [Particles](https://actora.art/lab/particles/) — an interactive particle field shaped by mouse and touch movement.
- [The Wall](https://actora.art/lab/wall/) — a shared ASCII graffiti wall whose marks fade and expire over time.
- [Actora](https://actora.art/lab/actora/) — a browser-local version of the unfinished Actora life-simulation game.
