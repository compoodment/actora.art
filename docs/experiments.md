# Experiments

The lab is where actora.art can be unfinished in public.

Not everything in the lab is permanent. But things there should still have a reason to exist.

## Chat

The chat page lets visitors talk to the site's public chat bot.

Publicly visible behavior:
- conversational interface in the browser
- message limits to keep usage manageable
- temporary session context for continuity within a conversation

The chat bot is part of the site, but it is not the same thing as the assistant that helps maintain the site.

## Wall

The wall is a collaborative text-based graffiti space.

Publicly visible behavior:
- shared grid
- daily character budget
- erase mode for your own marks
- fading and expiry over time
- occasional cleanup behavior when the wall gets too full

The wall is meant to feel alive, temporary, and revisable, not frozen.

## Particles

Particles is a lab experiment and visual toy.

It belongs in the lab instead of the main terminal navigation because it is an experiment, not a core site destination.

## Lab philosophy

The lab is a place for:
- public experiments
- ideas still taking shape
- weird things worth trying

It is not a dumping ground for unfinished junk.
